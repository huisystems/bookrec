## 故障排查

| 症状 | 原因 | 修法 |
|---|---|---|
| smoke exit 2（0 books） | cookie 路径有 bug 或 cookie 已失效 | 删 storage_state，重跑 `bookrec login`；若仍 0 books，看 stderr 的 `_handle_anti_scrape` 警告 |
| smoke exit 1（timeout/异常） | 网络/Playwright 异常 | 看 stderr 完整 stack；可能是 Playwright 版本问题 |
| `git push` 报 `could not read Username` | 未配凭据 | 走 Step 3 任一选项 |
| CI 报 `OIDC: failed to exchange token` | PyPI trusted publisher 配错 | 检查 owner/repo/workflow filename/environment 四项**完全一致** |
| CI 报 `403: project not found` | PyPI 上还没手动建 `bookrec` 项目 | https://pypi.org/manage/projects/ 手动建 |
| `pip install bookrec==0.2.3` 报 not found | PyPI 上 0.2.3 还没真发 | 等 Step 4 完整跑完再装 |
